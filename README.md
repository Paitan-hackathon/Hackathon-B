# サイト内容の説明
理科大発のプログラミングサークルである [Playground](https://playground-universe.com/undefined) 内で開催されたハッカソンにて作ったものです。「レジャー体験」というお題でサイト作成を行った際の制作物です

## 公への意図
フロントエンド実装などを勉強されてる方向けに、基本的なサイトにおける機能を盛り込んだサイトとなっております。フロントエンドを勉強する際にぜひご活用ください

# 主な機能の紹介
- 自動スクロールの実装：写真が自動で横に一定の時間間隔でスクロールされるようになっております
- スクロールによる自動フェードイン：画面を下にスクロールすると、一部のセクションが自動で順々に表示されるようになってます
- ダークモードの実装：よりサイトのカジュアルさにバリエーションを施すために、ダークモードも実装しております。タブで切り替えが可能になってます

### 注意
サイト内の情報は、すべて架空のものです。地図や電話番号などは、現実には存在しませんので、ご注意ください
